## Requirements

- ROS2 Foxy
- Gazebo
- px4-sitl (mavsdk)
- opencv
- mediapipe

## Install

```bash
colcon build
```

## Usage

```bash
ros2 run drone_cv cv_node
```

```bash
ros2 run drone_cv drone_node
```
